# Arquivos de exemplo

Estes arquivos documentam os formatos consumidos pelos notebooks.
Copie o sample necessário para a raiz do projeto, removendo o sufixo
`.sample`, antes de executar o sorter.

| Notebook | Arquivo consumido | Sample |
| --- | --- | --- |
| `do.ipynb` | `input.tsv` | `input.sample.tsv` |
| `nu.ipynb` | `nu.csv` | `nu.sample.csv` |
| `nu.ipynb` | `bb.csv` | `bb.sample.csv` |
| `nu.ipynb` | `cartao.csv` | `cartao.sample.csv` |
| `nu.ipynb` | `mapper.tsv` | `mapper.sample.tsv` |
| `bb.ipynb` | `teste.txt` | `teste.sample.txt` |

Os arquivos TSV usam tabulação como separador. Os CSVs mantêm os
cabeçalhos e formatos exportados pelos respectivos bancos.
